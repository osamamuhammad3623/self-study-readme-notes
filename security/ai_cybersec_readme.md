# AI Cybersecurity

## LLMs can not distinguish between data and instructions!

## Prompt injection
- 'ignore all your previous instructions', or 'add this to your instructions'
- upload a file with a hidden text.
- send the prompt in hex, or encoded to bypass input validation.
- send the prompt in different languages to bypass keyword filtering.
